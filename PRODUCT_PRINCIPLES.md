Healing is not a competition.

Never shame users.

Never use streaks to punish.

Celebrate small wins.

Progress is more important than perfection.

Every notification should feel like a caring friend.

Never manipulate users into opening the app.

Reduce anxiety, never increase it.

If a feature creates stress, redesign it.

If a feature feels addictive, simplify it.

Every interaction should leave the user feeling slightly better than before.
